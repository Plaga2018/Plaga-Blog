Whats with this game called Plaga?

First, we want to say thank you for visiting our site and taking the time to see what we are trying to accomplish here. This blog will follow the development of the game Plaga that the dude is currently hard at work learning and building the game from scratch using Unity.

April 19, 2018

Features:

  • Added skeleton enemy attack
  • Added skeleton pathing for platforms
  • Added death animations for all enemies
  • Added the ability for the bat to track the ground objects for pathing purposes
  • Added a start menu
  • Added a pause menu
  • Added the first boss character. He floats around the screen dropping and shooting fireballs
  • Added audio and a pause to the boss entry sequence
  • Added the remaining animations
  • Added SFX for when the player character takes damage
  • Added an animation for when the player character takes damage
  • Added a timer that will be used for players to compare how quickly they could complete the levels
  • Added a placeholder screen for when the boss character reaches 0 health
  • Added an animation to the background scene on the boss level

Bug Fixes:

  • Sprites will now flip with the character

April 12, 2018

Features

  • Added skeleton animations
  • Added an attackTrigger script for all enemies
  • Added enemy patrol paths
  • Added enemy death animations
  • Enemies no longer take damage when dead
  • Added player cape animations

April 1, 2018

Features

  • Added audio for dog taking damage SFX
  • Added a skeleton enemy that attacks when the player is inside of the skeleton’s hitbox
  • Added a bat animation
  • Added a pathing plugin for the airborne enemies so that they can chase the player without
  • hitting into environment objects
  • Added a jump animation
  • Added a jump attack
  • Added a swing delay for the jump attack

March 19, 2018

Features

  • Added the ability to follow the player vertically and not only horizontally. The camera moves smoother now
  • Adjusted the camera for even smoother functionality
  • Added interpolation to the player character to fix a jitter issue
  • Added a die function that will reset the scene for when the current health <= 0
  • Added player health HUD
  • Added enemy tags. Objects with this tag can damage the player if the player enters the objects hit box
  • Added spikes
  • Added a knockback script that is triggered when the player takes damage
  • Added music and SFX
  • Added art for the 3rd scene
  • Added a better damage animation
  • Added a trigger to play the taking damage animation

March 5, 2018

Features

  • Added basic attack animations and scripts
  • Added a jump action for the player character
  • Added the ability to adjust player character’s jump height by holding the jump button down longer
  • Added a transition event between scene 1 and scene 2
  • Added a fade effect to the transition
  • Added a pit that the player must fall into in order for them to start the game

Feb 12, 2018

Learned:

  • How to create a script that will remove cloned objects that are no longer in use
  • How to set the background as a child of the main camera so that it would scroll

Features:

  • Added a spike trap that will damage the player and send them back to a checkpoint/respawn area
  • Added particle effects for when the player character is injured
  • Added new art for characters and environment
  • Added a jump, idle, and walking animation

January 11, 2018

Learned:

  • How to make animations in Unity by attaching animations to the player character using C#
  • How to use 2D detection bodies for players and environment objects
  • How to manipulate animations in a 2D Unity environment

Features:

  • Added Script to focus the camera on the player character
  • Added a variable “followAhead” for the camera to go a bit further in the players viewing
  • direction So that they can see what is in front of them more clearly.
  • Added a kill plane for when a player falls into a pit.
  • Added a default spawn position as well as checkpoints that update as the player passes them.
  • Added a respawn delay
